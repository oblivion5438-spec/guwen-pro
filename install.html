iPhone 必須用 Safari 瀏覽器才能安裝，Chrome 不支援。

1
用 Safari 開啟顧問PRO
確認是用 Safari 瀏覽器（不是 Chrome 或 LINE 的內建瀏覽器）
🌐
網址列輸入：
guwen-pro.vercel.app
2
點底部的「分享」按鈕
畫面最下方中間有個方框加箭頭的圖示，點它
📤
在 Safari 底部工具列
中間的那個按鈕
3
往下滑找「加入主畫面」
在跳出的選單中往下滑，找到「加入主畫面」並點它
➕
選單裡找「加入主畫面」
（可能需要往下捲動）
4
點右上角「新增」
確認名稱為「顧問PRO」，點右上角「新增」完成安裝
✅
名稱確認後點「新增」
桌面就會出現圖示
🎉

安裝完成！

回到桌面找到顧問PRO 圖示
點開就是全螢幕 APP 模式

✅

Android 使用 Chrome 瀏覽器，瀏覽後會自動提示安裝，也可以手動操作。

1
用 Chrome 開啟顧問PRO
確認是 Chrome 瀏覽器，不是其他內建瀏覽器
🌐
網址列輸入：
guwen-pro.vercel.app
2
等待安裝提示橫幅
登入後畫面下方會自動出現「安裝顧問PRO」橫幅，點「安裝」即可
📲
下方橫幅出現後
直接點「安裝」
或
手動安裝（沒看到橫幅時）
點右上角三個點 → 往下找「加入主畫面」或「安裝應用程式」→ 點確認
⋮
右上角「⋮」→
「加入主畫面」
3
點「安裝」確認
跳出確認框後點「安裝」，桌面就會出現顧問PRO 的圖示
✅
確認安裝後
桌面出現 APP 圖示
🎉

安裝完成！

從桌面點開顧問PRO
全螢幕 APP 模式，沒有網址列

🚀 立即開啟顧問PRO
安裝後從桌面圖示開啟效果更好
